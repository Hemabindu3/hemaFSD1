1. Ordered List

An ordered list is a list where the items are numbered.

  1. First item
  2. Second item

2. Unordered List

An unordered list is a list where the items are bulleted.

  • Apples
  • Bananas

3. Nested Lists

A nested list is a list that contains another list inside it. Here's an example:

  • Fruits
    • Apple
    • Banana
  • Vegetables
    • Carrot
    • Spinach

4. Ordered List in an Unordered List

We can place an ordered list inside an unordered list:

  • Breakfast
    1. Eggs
    2. Toast
  • Lunch
    1. Rice
    2. Chicken

5. Definition List

A definition list is used to display pairs of terms
